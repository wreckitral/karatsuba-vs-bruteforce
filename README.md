# Analisis Algoritma

Tugas akhir analisis algoritma, 
analisis perbandingan algoritma brute force dan divide and conquer implementasi dari algoritma karatsuba pada perkalian integer.

## How to run
type command 
`cd karatsuba`
`go run main.go`

## How to test 
type command
`cd karatsuba`
`go test -bench=.`
